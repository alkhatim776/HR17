Leave - Auto annual leave allocation plus some attributes on leave type

Description

This module adds a set of attributes to the leave type and leave request. Also set a daily accrual leave allocation for every employee according to the saudi arabia labor law. You can also see total leave, exhausted and remaining balance of every Employee.

Requirements

  • Time Off (hr_holidays)

  • Employees (hr_employee_extension)

  • Employee Contracts (hr_contract)

Key Features

  • Adds a serial number for each Leave request.

  • Adds a Calculation Type [Working Days, Calendar Days] in Leave Types.

  • Adds annual leave type and annual leave days per year in the employee contract.

  • Adds the Leave Balance menu where you can see total, exhausted and remaining balance of every employee.

  • Allow adding leave type that is related to specific gender.

  • Prevent saving leave request without Attaching a supporting document.

  • Auto leave allocation based on employee leave type and days per year
